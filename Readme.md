# Freedom micro-framework

## Easy to configure your custom project settings with Config module

- path ```/config/<config_name>.php```
- get ```Config::getInstance()->get('<config_name>')```
- syntax in <config_name>.php ```
  return [
  "param 1" => "value1",
  "param 2" => "value2",
  ]```
